# Tipos de datos
## Números enteros
- `byte`: 8 bits. -128;128
- `short`: 16 bits. -32768;32768
- *`int`*: 32 bits. -2.147.483.648;2.147.483.648
- `long`: 64 bits.
## Números racionales
- `float`: 32 bits
- *`double`*: 64 bits
